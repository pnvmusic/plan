## 🔄 วิธีอัปเดตเว็บในอนาคต

### กรณีที่ 1 — แค่เพิ่ม/แก้ข้อมูล (เพลง, งาน, ค่าใช้จ่าย ฯลฯ)
**ไม่ต้องทำอะไรพิเศษ** — ทำในเว็บได้เลย บันทึกลงฐานข้อมูลทันที ทุกคนเห็นพร้อมกัน

### กรณีที่ 2 — แก้โค้ด / เพิ่มฟีเจอร์ / เปลี่ยนหน้าตา
เปิด Terminal แล้วเข้าโฟลเดอร์โปรเจกต์ก่อนเสมอ:
```
cd ~/Sandbox/"pnv Plan WebApp"/museflow-app
```

จากนั้นรัน 3 คำสั่ง:
```
git add .
git commit -m "อธิบายสั้นๆ ว่าแก้อะไร"
git push
```

GitHub จะ build + deploy ใหม่อัตโนมัติ (~1-2 นาที) ดูสถานะที่หน้า Actions
ให้ขึ้นเครื่องหมายเขียว ✅ แล้วรีเฟรชเว็บ

### ทดลองก่อน push (ไม่บังคับ แต่แนะนำ)
ก่อน push ลองรันบนเครื่องดูว่าโอเคไหม:
```
npm run dev
```
เปิด http://localhost:5173/ ดู ถ้าโอเคค่อย push

## ➕ เพิ่มสมาชิกใหม่ในทีม

ตอนนี้ระบบไม่เปิดให้สมัครเอง (ผู้ดูแลสร้างบัญชีให้) วิธีเพิ่มคน:
1. เข้า Supabase Dashboard → **Authentication** → **Users** → **Add user**
2. ใส่ email แบบ `ชื่อ@museflow.local` (เช่น `mind@museflow.local`) + รหัสผ่าน
3. ติ๊ก **Auto Confirm User**
4. คนใหม่จะเป็น Team Member อัตโนมัติ — ถ้าอยากเปลี่ยนเป็น Manager/Admin
   ให้ Admin เข้าเว็บ → เมนู "ทีม & สิทธิ์" → เปลี่ยน role ได้

## 🆘 แก้ปัญหาที่เจอบ่อย

**Login ขึ้น error / เข้าไม่ได้**
- เช็ค username/รหัสผ่านถูกไหม (username ไม่ต้องใส่ @ อะไร พิมพ์แค่ `pnv`)

**push แล้ว Actions ขึ้นแดง ❌**
- คลิกเข้าไปดูข้อความ error → มักเป็นพิมพ์โค้ดผิด
- ถ้าแก้ไม่ได้ ส่งภาพ error มาถามได้

**เว็บเปิดมาขาว/ไม่มีอะไร**
- รอ deploy ให้เสร็จก่อน (เขียว ✅ ที่หน้า Actions)
- เคลียร์แคชเบราว์เซอร์แล้วลองใหม่
